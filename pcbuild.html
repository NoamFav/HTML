🚀 Ultimate PC Build Guide

CUDA-Ready Gaming Workstation • Dual Boot • Future-Proof

≈€2k starter Build v1 — CUDA-ready
  • Case:
    Corsair 7000D Airflow (black or white) — full tower, E-ATX, 420/360 mm rad support, 4-slot GPU clearance.
  • PSU:
    Corsair RM1000e (ATX 3.0, 1000 W) — quiet now, headroom for future GPU.
  • CPU:
    AMD Ryzen 7 7800X3D (AM5).
  • Cooler:
    Thermalright Peerless Assassin 120 SE (air) — or Arctic Liquid Freezer III 360 (AIO) if you want it silent under load.
  • Motherboard:
    MSI MAG B650 Tomahawk WiFi (ATX, 4× DDR5 DIMMs, multiple M.2).
  • RAM:
    128 GB (2×64) DDR5-5600 EXPO — Crucial Pro CP2K64G56C46U5.
  • GPU:
    NVIDIA GeForce RTX 4070 SUPER (any 2–3-fan AIB).
  • Storage:
    Samsung 990 PRO 4 TB (PCIe 4.0 NVMe).
  • Case fans:
    3× quality 140 mm PWM (e.g., Arctic P14 PWM or Noctua P14 redux) to fully populate the 7000D front/top.
≈€6–7k upgrade Build v2 — the beast
  • CPU:
    swap to AMD Ryzen 9 9950X.
  • GPU:
    swap to NVIDIA GeForce RTX 5090 (4-slot; plan front intake/top exhaust).
  • PSU:
    swap to Corsair RM1200x SHIFT (ATX 3.1, native 12V-2×6 cable).
  • RAM:
    add the same kit as v1 → 256 GB total (4×64).
  • Storage:
    add second Samsung 990 PRO 4 TB (use as games/datasets/scratch).
  • Cooler:
    move to 360 mm AIO (if not already) — e.g., Corsair iCUE H150i / Arctic LF III 360.
  • Extras:
    GPU anti-sag bracket; optional 10 GbE PCIe card; extra 140 mm fans if needed.
Optional "Endgame" adds (later, if you want)
  • More NVMe via PCIe M.2 adapter card (adds 2–4 drives).
  • Third Samsung 990 PRO (backups/scratch) + external NVMe for offsite borg/borgmatic.
  • 2.5/10 GbE switch + NAS if you start hoarding datasets.
Clean dual-boot layout (2× NVMe)
💾 Drive 1 (Arch):
EFI (512 MB, FAT32)
btrfs root with subvolumes @, @home, @snapshots (mount with ssd,compress=zstd:3,noatime).
💽 Drive 2 (Windows 11):
let Windows create its EFI + NTFS; keep whole disk NTFS.
Boot Strategy: Boot via BIOS picker (no shared bootloader drama). Mount the Windows NTFS in Arch when needed.

Cable & small stuff (don't forget)
  • Pre-applied paste on many AIOs; otherwise grab Thermal Grizzly Kryonaut.
  • Extra 12V-2×6 GPU cable comes with the RM1200x SHIFT (use native, no adapters).
  • Quality USB-C (40 Gbps) and DisplayPort 1.4/2.1 cables for your monitors.
Summary (how you'll use this)
  • Buy v1 now (fits your budget, zero compromises for gaming + CUDA work).
  • Swap to v2 later: CPU, GPU, PSU, add RAM + second NVMe — everything else stays.
  • Case/airflow/clearances are already sized for the 5090 + 360 mm cooling.
  • Arch + Hyprland on NVMe #1, Windows on NVMe #2, pick at boot. Smooth.

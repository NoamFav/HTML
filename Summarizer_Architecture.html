üßπ Phase 2: Email Body Extraction & Cleaning
1
Header Removal
Strips standard email headers: Message-ID, From, To, Subject, Date, CC, BCC, MIME headers
2

Removes Enron-specific metadata: X-Folder, X-Origin, X-FileName entries
3
Forward/Reply Removal
Eliminates forwarding chains and reply metadata using pattern matching
4
Email Address Scrubbing
Removes all email addresses from CC, BCC, and body content
5
Whitespace Normalization
Cleans excessive newlines and normalizes text formatting
üîç Phase 3: Text Parsing & Analysis
1
Plaintext Parsing
Uses sumy's PlaintextParser to convert cleaned email into structured document
2

Applies NLTK English tokenizer to split text into sentences and words
3
Document Structure Analysis
Analyzes sentence structure and importance for optimal summarization
üß† Phase 4: LSA Summarization Process
1
LSA Algorithm Initialization
Initializes Latent Semantic Analysis summarizer for topic extraction
2
Sentence Scoring
Calculates semantic importance scores for each sentence using SVD
3
Top Sentence Selection
Selects configurable number of highest-scoring sentences (default: 3)
4
Summary Generation
Combines selected sentences into coherent summary text
üìä Phase 5: Output Processing & Error Handling
1

Formats selected sentences with proper line breaks and structure
2
Exception Handling
Handles ValueError, AttributeError, TypeError, and RuntimeError gracefully
3

Returns "Unable to generate summary" message for failed processing
4
Final Output
Delivers clean, readable summary or appropriate error message

üîç Regular Expression Patterns
r"^(Message-ID|From|To|Subject|Date|Cc|Bcc|Mime-Version|Content-Type|...):.*$"
Removes standard email headers
r"X-Folder:.*|X-Origin:.*|X-FileName:.*"
Eliminates Enron-specific metadata
r"(-{2,}|={2,})\s*(Original Message|Forwarded by).*"
Removes forwarding and reply chains
r"[\w\.-]+@[\w\.-]+\.\w{2,3}"
Strips email addresses from content
r"\n\s*\n+"
Normalizes excessive whitespace
